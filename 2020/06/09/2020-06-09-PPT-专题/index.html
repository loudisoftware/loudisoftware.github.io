发表于 2020-06-09 更新于 2020-09-16

审美观

从海报、视觉、画册、平面设计、演示中学习，增加自己的审美观。

文字修饰

  1. 为文字增加线框
  2. 倒影效果 （形状倒影）
  3. 镂空效果（合并形状剪除）
  4. 三维旋转 (颜色渐变+旋转)

形状

  1. 合并形状
  2. 通过形状画小图标
  3. 通过形状分割版面 （开放路径）
  4. 版面透明框
  5. 强度内容，增加背景形状。

配色

  1. 颜色太多
  2. 取色器获取配色方案
  3. 主色调，背景色，字体颜色
  4. 通过色调值变化成其他主题的颜色

图标优化

  1. 增加图标专业度
  2. 降低信息量
  3. 修饰布局

表格修饰

  1. 建立主色调
  2. 建立边框线
  3. 规律性底纹

版面设计

  1. 对齐
  2. 对比（颜色，大小，形状，距离）
  3. 亲密性（相关的内容距离更近）
  4. 重复（风格统一）

页面框架

  1. 分割（上下，左右）
  2. 分裂（多个区域）
  3. 居中（中间放文字）
  4. 包围
  5. 对称
  6. 杂志
